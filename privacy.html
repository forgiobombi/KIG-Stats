Privacy Policy

Last Updated: July 2026

Your privacy and digital alignment are top priorities. This disclosure clearly states how data is handled by the Kig Stats bot application.

1. Data We Access & Collect

  • Guild Configuration Records: We store customized server prefixes, embed formatting style modes, and target logging channel targets within our safe localized database handles to make the system operate.
  • User Settings: Preferred toggle options regarding command rendering (such as preferred graphic vs embed display methods) are remembered.
  • Transient System Lookups: When checking statistics, your Minecraft UUID handles are temporarily referenced against public APIs to compile profile visualizations.

2. Hosting Infrastructure

This policy portal is deployed as a GitHub Pages application. GitHub may automatically log visitor IP parameters to address general infrastructure stability and fulfill legal compliance protocols.

3. Data Retention & Deletion

Guild configuration parameters are retained for operational continuity. If the bot is removed from a target server guild environment, associated customized prefix properties are subject to purging logic routines. To request manual user parameter clearing, contact developer forgiobombi.
